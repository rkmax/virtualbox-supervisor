virtualbox-supervisor

Virtualbox Supervisor es una herramienta similar a XM para Xen que permite controlar las maquinas virtuales y mirar sus estados.

el comando incluye un autocompletar de cada una de las opciones abajo listadas. ver (archivo _vs).

actualmente es un envoltorio para el comando VBoxManage

comandos

status

permite ver el consumo de Memoria y CPU de cada maquina virtual y el total.

top

igual que status pero mantiene actualizada la pantalla.

start

inicia una maquina virtual en modo headless

stop

detiene una maquina virttual mediante acpipowerbutton de virtualbox.

force_stop

apaga por completo una maquina virtual.
